# 💻 Practice

A collection of my solutions to various programming and algorithmic problems.

This repository serves as a personal archive of interesting coding tasks, university exercises, and algorithm implementations. The problems are **independent of one another**, and each is organized in its own folder.

## 📂 Repository Structure

Each problem has its own directory containing:

- `solution.py` — the Python implementation.
- `README.md` — a description of the problem, and approach.
- `picture.png` _ if included in the question of an output

```
Practice/
│
├── Caesar Cipher V2/
│   ├── caesar cipher v2.py
│   └── README.md
│
├── Caesar Cipher is hacked/
│   ├── Caesar Cipher is hacked1.py
│   └── README.md
│
├── trapping-rain-water/
│   ├── rain.py
│   └── README.md
│   └── taskrain.png
│
└── ...
```

## 📖 Current Problems

| Problem | Description |
|---------|-------------|
| 🔐 Caesar Cipher V2 | Encrypt a message using a Caesar Cipher with an increasing shift for each letter. |
| 🔓 Caesar Cipher is hacked | Recover the original message using a known plaintext character and the encryption rules. |
| 🌧 Trapping Rain Water | Calculate the total volume of rainwater trapped between terrain elevations. |
